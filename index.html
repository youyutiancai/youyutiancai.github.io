About me

I am currently a PhD candidate at Purdue University, where I started my PhD program since 2021, advised by Professor Voicu Popescu. I am interested in XR research, especially VR haptics. My take for the future VR haptics is an immersive world that enables the user to touch *anything* within their reach, a dream shown in the movie Ready Player One. For the first step, I designed and built the first prototype of Encountered-type Haptic Device (ETHD) last year in the form of a Cartesian robot. I'm also a big fan of wearable haptic devices and will work on them in the future. Other than VR haptics, I also have first-author publications in other XR fields, such as Cloud VR and collaborate AR.

Selected work

VR haptics with ETHD

The first component of all-accessible system is a robot that serves as physical replica of stationary objects, especially those placed on stationary surfaces such as a table. Without an ETHD, a wearable device that can simulate the tactile feedback cannot properly halt the user's hand. As a result, the user's hand will penetrate through the object. I built this robot mainly for proving some of my thoughts and hypotheses.

CloVR: Fast-Startup Low-Latency Cloud VR

Speaking of cloud gaming for VR, one can only think of the traditional way of reading the user's input and sending computed frames from the server to the user. This approach works for PC games, but the roundtrip latency can easily nauseate the user. We came up with a novel method that transfers part of the virtual environment from the server to the client and renders the rest as a cubemap. This approach can maintain complete visualization and fast startup features for a large-scale 3D scene, while the local rendering with high framerate does not make the user sick.

Co-located AR Attention Guidance

In this project, I developed a system for a "guide" to point out a real-world reference point to a "tourist" using tablets with cameras. The guide annotates the point, sent to the tourist's tablet, and guides them to the target with zoom. It outperforms conventional augmented reality in localization speed and provides directional guidance without relying on visual details in challenging scenes.

Unlike a robot that can move objects around, a stationary object can only provide passive haptic feedback of virtual objects that are perfectly aligned. As a result, the same setup of a physical environment can only be used to offer haptic feedback for a specific virtual setup. The redirection algorithm can solve this problem by retargeting the user body or handheld stick (in our case) gradually when the user is approaching the physical object. I came up with a noval algorithm that not only ensures the synchronized virtual and physical contact at the end, but also enables shape mapping with smooth transition. That is, a physical object can provide haptic feedback for virtual objects in different shapes.
